Hva er CSS

CSS står for Cascading Style Sheets.
CSS er språket som bestemmer hvordan nettsiden skal se ut.
CSS styrer farger, skrifttyper, layout, avstander, størrelser osv.

På nettsiden til venstre har vi nå lagt på CSS, slik at nettsiden blir mer inbydende.
