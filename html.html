Hva er HTML

HTML står for HyperText Markup Language, og er et markeringsspråk.

HTML er språket som brukes til å lage strukturen på nettsider.
Det er med HTML man bestemmer nettsidens oppbygning.

HTML forteller nettleseren hva som er innholdet på nettsiden.
(overskrifter, tekst, bilder, lenker, video osv.)

HTML Koden

Til venstre er den samme nettsiden du så på forsiden, men kun laget med HTML.
HTML forteller hva som skal være med på nettsiden, men ikke hvordan det skal se ut.
