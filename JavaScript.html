Hva er JavaScript

JavaScript er språket som gir nettsider liv og interaktivitet.
JavaScript gjør at ting kan skje på siden (klikk, animasjoner, skjema, spill, beregninger)
JavaScript kan endre HTML og CSS mens siden kjører

Oppsummert

HTML gir innhold
CSS gir utseende
JavaScript gir liv og interaktivitet
